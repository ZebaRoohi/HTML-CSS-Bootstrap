Example of CSS position properties

Static positioning is the default behavior for HTML elements. Elements are positioned in the normal flow of the document. No special positioning properties are required.
Elements with relative positioning are positioned relative to their normal position in the document flow.You can use the `top`, `right`, `bottom`, and `left` properties to offset the element.
An element with position: absolute; is positioned relative to the nearest positioned ancestor,if an absolute positioned element has no positioned ancestors, it uses the document body, and moves along with page scrolling.
An element with position: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled
Sticky elements are initially in the normal flow and switch to fixed positioning when they reach a specified scroll position.You can use the `bottom`, and `right` properties to control the sticky position
